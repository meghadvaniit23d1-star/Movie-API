🎬 Movie API Project
📌 Overview

The Movie API Project is a web application built using Node.js, Express.js, and EJS that allows users to browse, search, and manage movie data. It demonstrates backend API development along with server-side rendering for dynamic web pages.

This project follows RESTful architecture and integrates frontend views using EJS templates to display movie information interactively.
-------
🚀 Features
🎥 View list of movies
🔍 Search movies by title, genre, or rating
➕ Add new movies
✏️ Update movie details
❌ Delete movies
📄 Dynamic UI using EJS templates
🔗 RESTful API endpoints
🛠️ Tech Stack
Backend: Node.js, Express.js
Frontend: EJS (Embedded JavaScript Templates)
Database: MongoDB (mongoose).


⚙️ Installation & Setup
Clone the repository
git clone https://github.com/your-username/movie-api.git

Install dependencies
npm install

Run the server
npm start

Open in browser
http://localhost:3000
------
🔗 API Endpoints
Method	Endpoint	Description
GET	/movies	Get all movies
GET	/movies/:id	Get single movie
POST	/movies	Add new movie
PUT	/movies/:id	Update movie
DELETE	/movies/:id	Delete movie
